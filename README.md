# Task Board Docs

This repository contains the source files for the Task Board plugin documentation website.

The docs are hosted here: [Task Board Docs](https://tu2-atmanand.github.io/task-board-docs/)

The official **Task Board** Obsidian plugin can be found here: [Task Board](https://github.com/tu2-atmanand/Task-Board)

## How to Contribute

If you would like to help update any content from the website or have any suggestions, please check out: [How to contribute to documentation](/docs/Advanced/HowToContributeToDocs.md)

I am personally developing this docs inside VS Code using the [ExplorerSorter extension](https://github.com/MeydanOzeri/ExplorerSorter) which helps to see the same files structure as it is in the published docs. The extension uses `.order` files present inside each of the folder in this project, to sort the files/folders in a custom order.
