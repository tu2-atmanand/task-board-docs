# Major Upcoming Features

Many features that were previously on this list have now been implemented! Here's what's still coming:

## Planned Features

- **Different types of view :** Different kinds of views such as inbox, list, table, etc.
- **Store task activity :** Convert any inline-task/task-note into a tb-note to store every single activity related to the task to log your progress. Like a diary for each of your important task.
- **File per Board :** Store each of your board as a `.taskboard` file and create as many different types of views required inside the board file.
- **Temporary Task :** User will be able to add a temporary task which is not required to be saved in any file.
- **Advanced Search Filters :** Search tasks by tags, dates, priority, and other properties.
- **Canvas Integration :** Features related to canvas support.
