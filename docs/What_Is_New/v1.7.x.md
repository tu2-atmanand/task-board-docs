## Task Board v1.7.0 üéâ

> YouTube video for this release : https://youtu.be/AEkX8mOHGeE

### New Features üéÅ

- **More task formats support :** Almost all task formats such as indented tasks and tasks in callouts are now supported. | #133
- **Config Import/Export :** Now users can export their setting to take a backup and import again easily. } #354
- **Hide properties :** Hide selective task properties Reading as well as Live editor mode. | #111
- **Auto add unique-id :** The plugin will automatically add unique Id to your tasks. Necessary for various other features. | #254
- **Parent-Child linking :** Add child-tasks to your parent task as a dependency property and restrictions } #211 & #380 
- A new setting to restrict completion of parent-task when child-tasks/sub-tasks are incomplete. | #380
- **Store last view :** Now the plugin will remeber your last viewed board and the view type (Kanban or Map view).
- **Double click card actions :** A new setting to select the action you want for card double click. | #267
- All **wildcards** for tag setting wherever applicable to improve usability for nested tags | #343 & #372
- **QMD files support :** Tasks from QMD files will be now supported. | #331
- Use **Tasks plugin's edit task modal** through the edit button of task item card. | #339
- A **reload Obsidian** notice with a button will be shown whenever applicable setting has been changed. | #368
- Show last modified/added tasks on top of the column as a default sorting criteria until sorting feature is implemented. | #352

### Experimental Features
These features can be enabled using the **Experimental features** setting.

- **Task Notes :** Show normal notes as task cards to add a reminder to your notes and track them on the Kanban board.
- **Map view :** Spread out your tasks on a canvas, organize them efficiently and connect parent-child tasks to create a chain of tasks to manage your complex projects with ease.
