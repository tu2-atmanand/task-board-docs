# How to edit a task

There are two ways to edit an existing task:
1. [Edit Task from File](#1-from-the-markdown-file) : Edit the item within the markdown file itself.
2. Via the [Task Editor component](./docs/6_Features/2_Task_Editor.md) : This can be done from your board, even without opening the source markdown file.

## 1. From the markdown file

This is the manual method, on how you use to edit the content of the task from the parent markdown file. You will have to open the markdown file, which contains the task and edit any property of the task you want.

A [new setting](./HowToUseGlobalSettings.md#edit-button-mode) provides you four options to open the parent markdown note using the button itself and make it easy to edit the task.


After you done editing, you can simply switch your focus from the editor to the [Task Board View](./docs/7_Components/Task_Board_Pane.md) and your task will be instantly refreshed on the Task Board.

## 2. Using Task Editor

This feature exists to allow you to directly edit the content and properties of a task, directly from your board, without the need of opening the parent markdown file. All changes made within this component will be reflected in the markdown file.

**Step 1:** Find the task you would like to edit and click the **Edit Task** button/icon. This will open the **Edit Task Window**.

![Editing Task Panel](..//assets/EditTaskWindow.gif)

**Step 2:** Edit any field you would like from the appropriate input field. You can also add values to the empty fields, add descriptions, add more sub-tasks, etc.

**Step 3:** **Select the "Save" button to save changes made**. _(If not, the user will be given the opportunity to discard changes or go back and save)_

> To learn more about the **Task Editor**, including detailed explanations of each field, check here: [Task Editor](./docs/6_Features/2_Task_Editor.md).
