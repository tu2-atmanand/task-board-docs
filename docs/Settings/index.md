# Plugin Settings

Task Board plugin's settings are organized into tabs under the plugin's setting. Each tab controls a different part of plugin behavior.

Please note that these are the global settings which will be applied to the whole plugin. But, you also have some more settings which are board or view specific, which you can find inside the [Board config modal](/docs/How_To/Use_Board_Settings.md).

Please find below all the setting tabs...

## General

Under this tab you will find some miscellaneous settings which cannot be grouped under the below sections. And also these are very powerful settings.

Read more here : [General settings](./General.md).

## All UI

Everything related to a the UI of Task Board can be configured under this setting's section. You will find different setting to customize the UI of the task card, the width of the columns, etc.

Read more here : [All UI settings](./All_UI.md).

## Properties

Under this section you will find setting where you can configure different aspects related to the task properties supported by this plugin. In future, you will also be able to configure your own properties here.

Read more here : [Task properties settings](./Properties.md).

## Inline tasks

In this section you will find all the setting specific to the inline-tasks type of tasks.

Read more here : [Inline tasks settings](./Inline_tasks.md).

## Task notes

In this section you will find all the setting specific to the task-notes type of tasks.

Read more here : [Task notes settings](./Task_notes.md).

## Map view

This section contains settings which are related to the [Map view feature](/docs/Features/Dependency_Map.md).

Read more here : [Map view settings](./Map_view.md).

## Automations

Under this section you will find multiple settings which can be used to add some automation to your task management workflows. You will also find settings to enable the [integration with other plugins and external services/apps](/docs/Features/Plugin_Integrations/Overview.md).

Read more here : [Automations settings](./Automations.md).

## Formats

This section contains all the settings which will help you to choose between different formats available for different properties. You can also configure your custom formats for properties like dates.

Read more here : [Formats settings](./Formats.md).
