# Task Formats

At present Task Board supports two kinds of tasks in Obsidian : 
- [Inline Task](./Inline_Task.md)
- [Note per Task](./Note_per_Task.md)

Both types of tasks will work the same way inside Task Board, the only difference is how they are stored inside Obsidian.
