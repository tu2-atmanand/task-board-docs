# Undated

This column will contain only the tasks which do not have the specific date set to them, which you have selected in the below setting.

## Properties

### **Column Name**

To create this column you only need to enter the name of the column, which will be visible on the column bar. If you want to see all the tasks, which are not been scheduled yet, then you can create this column and schedule them easily.


### **Date type**

The type of the date, similar to the [universal date](/docs/8_Settings/8_Formats.md#universal-date). When you will create fresh new column, the date which you have set inside the universal date setting will be applied here, which you can easily change. This specific date property which you going to select here, should **NOT** be present on the task, for that task to appear inside this column.
